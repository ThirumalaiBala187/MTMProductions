Question 1
What is JavaScript primarily used for?

Correct! JavaScript is primarily used to make web pages interactive. While it can also be used for server-side programming (Node.js), database interactions, and even desktop applications (Electron), its primary purpose is adding interactivity to websites.

Question 2
Which of the following is a valid way to declare a variable in JavaScript?

Correct! In JavaScript, you can declare variables using let, const, or var keywords. The let keyword was introduced in ES6 and is now the recommended way to declare variables that will change.

Question 3
What will the following code output? console.log(2 + "2");

Correct! In JavaScript, when you use the + operator with a string and a number, JavaScript converts the number to a string and performs string concatenation. So 2 + "2" becomes "2" + "2" which equals "22".

Question 4
Which of the following is NOT a JavaScript data type?

Correct! JavaScript does not have a specific "Character" data type. In JavaScript, individual characters are represented as strings with a length of 1. The standard data types in JavaScript include String, Number, Boolean, Object, Undefined, Null, Symbol, and BigInt.

Question 5
Which method is used to add elements to the end of an array in JavaScript?

Correct! The push() method adds one or more elements to the end of an array and returns the new length of the array. For example: let arr = [1, 2]; arr.push(3); results in arr being [1, 2, 3].
