1 Lists: Creating, Retrieving, and Accessing

Lists in Python are ordered, mutable collections of items. They can hold various data types and are one of the most versatile data structures.

1

Creating a list:

fruits = ["apple", "banana", "cherry"]
2

Accessing elements (indexing starts at 0):

print(fruits[0]) # Output: apple print(fruits[-1]) # Output: cherry (negative indexing)
3

Modifying and retrieving:

fruits[1] = "blueberry" # Modify print(fruits) # Output: ['apple', 'blueberry', 'cherry']
Did you know?

Lists can contain mixed data types, e.g., [1, "hello", True].

2 Tuples

Tuples are similar to lists but are immutable, meaning their elements cannot be changed after creation. They’re useful for fixed collections of data.

1

Creating a tuple:

coordinates = (10, 20)
2

Accessing elements:

print(coordinates[0]) # Output: 10
3

Trying to modify (this will fail):

coordinates[0] = 15 # Raises TypeError
Why Use Tuples?
  • Faster than lists due to immutability.
  • Used for data that shouldn’t change, like constants.

3 Dictionaries: Key-Value Pairs

Dictionaries store data as key-value pairs, allowing you to associate unique keys with values for efficient lookups.

1

Creating a dictionary:

student = {"name": "Alice", "age": 20, "grade": "A"}
2

Accessing values:

print(student["name"]) # Output: Alice
3

Modifying and adding pairs:

student["age"] = 21 # Modify student["id"] = 123 # Add new pair print(student)
Pro Tip

Use dict.get(key, default) to safely retrieve values with a fallback if the key doesn’t exist.

4 List and Dictionary Comprehensions

Comprehensions provide a concise way to create lists and dictionaries using loops and conditions in a single line.

1

List comprehension:

squares = [x**2 for x in range(5)] print(squares) # Output: [0, 1, 4, 9, 16]
2

List comprehension with condition:

even_squares = [x**2 for x in range(5) if x % 2 == 0] print(even_squares) # Output: [0, 4, 16]
3

Dictionary comprehension:

square_dict = {x: x**2 for x in range(5)} print(square_dict) # Output: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
Did you know?

Comprehensions are not only concise but often faster than traditional loops!

5 Sets in Python

Sets are unordered collections of unique elements, ideal for removing duplicates and performing mathematical operations like unions and intersections.

1

Creating a set:

numbers = {1, 2, 3, 3, 4} print(numbers) # Output: {1, 2, 3, 4} (duplicates removed)
2

Adding elements:

numbers.add(5) print(numbers) # Output: {1, 2, 3, 4, 5}
3

Set operations:

set1 = {1, 2, 3} set2 = {2, 3, 4} print(set1 & set2) # Intersection: {2, 3} print(set1 | set2) # Union: {1, 2, 3, 4}
Pro Tip

Use sets to quickly check for membership with in, e.g., 3 in numbers.
