1 Installing Python

Python is a versatile programming language used for web development, data analysis, artificial intelligence, and more. Before we can start coding, we need to install Python on your computer.
The latest version of Python includes the interpreter (which runs your code) and pip (a package manager for installing additional libraries).

1

Visit the official Python website at python.org/downloads.. Click the download button for the latest version (3.11.x or later is recommended)

2

Run the installer that downloads.. Important: Check the box that says "Add Python to PATH" before clicking Install

3

Verify the installation by opening Command Prompt (Windows) or Terminal (Mac/Linux) and typing:

python --version
Python 3.11.4

You should see the version number displayed, confirming Python is installed correctly.

Did you know?

Python was created by Guido van Rossum in the late 1980s. The language name doesn't come from the snake, but from Monty Python's Flying Circus, a British comedy series that van Rossum was a fan of!

2 Your First Python Program

Now that Python is installed, let's write our first program - the classic "Hello, World!" example. This simple program will display text on the screen and confirm that everything is working properly.

1

Open a text editor (like Notepad, TextEdit, or VS Code)

2

Type the following code:

# My first Python program: print("Hello, World!") # Try adding your own message: print("I'm learning Python!")
3

Save the file as hello.py

4

Run your program by opening Command Prompt/Terminal, navigating to where you saved the file, and typing:

python hello.py
Hello, World! I'm learning Python!
Understanding Your First Program
  • Lines that start with # are comments. They're ignored by Python but help humans understand the code.
  • The print() function displays text on the screen.
  • Text inside quotation marks "like this" is called a string.

3 Choosing a Development Environment

While you can write Python code in any text editor, dedicated development environments make coding easier with features like syntax highlighting, code completion, and debugging tools.

1

Visual Studio Code (VS Code): A lightweight, versatile editor with excellent Python support through extensions. Free and open-source. Install from: code.visualstudio.com

2

PyCharm: A comprehensive Python IDE with advanced features. Available in free Community and paid Professional editions. Install from: jetbrains.com/pycharm

3

Jupyter Notebooks

An interactive environment perfect for data science, where code, visualizations, and explanations live together.

Install via command: pip install notebook

Pro Tip

VS Code is an excellent choice for beginners, offering a good balance of simplicity and power. After installation, be sure to add the Python extension for the best experience.

4 Variables and Data Types

In Python, variables store data, and each variable has a type that determines what kind of data it holds. Python supports various data types such as numbers, text, and booleans.

1

Creating variables:

name = "Python" # String
2

Printing variables:

print(name)
Python
3

Checking variable types:

print(type(name)) # Output: print(type(age)) # Output:
Did you know?

Python is dynamically typed, meaning you don’t need to declare variable types explicitly!

5 Basic Operators in Python

Operators perform operations on variables and values. Python supports arithmetic, comparison, and logical operators.

1

Arithmetic Operators:

a = 10 b = 5 print(a + b) # Addition
2

Comparison Operators:

print(a > b) # Greater than print(a == b) # Equal to print(a != b) # Not equal to
3

Logical Operators:

x = True y = False print(x and y) # Logical AND print(x or y) # Logical OR print(not x) # Logical NOT
Pro Tip

Use parentheses `()` to control operator precedence, just like in math.
