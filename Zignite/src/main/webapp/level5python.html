1 Reading and Writing Files

Python provides built-in functions to read from and write to files, making it easy to handle text, data, or configurations.

1

Writing to a file:

with open("example.txt", "w") as file: file.write("Hello, Python!")
2

Reading from a file:

with open("example.txt", "r") as file: content = file.read() print(content) # Output: Hello, Python!
3

Appending to a file:

with open("example.txt", "a") as file: file.write("\nLearning is fun!") # File now contains two lines

2 Using Modules

Modules are pre-written Python files you can import to use their functions, classes, or variables in your code.

1

Importing a built-in module:

import math print(math.sqrt(16)) # Output: 4.0
2

Importing specific functions:

from math import pi print(pi) # Output: 3.141592653589793
3

Using an alias:

import random as rnd print(rnd.randint(1, 10)) # Output: Random number between 1 and 10
Why Use Modules?
  • Reuse code without rewriting.
  • Keep your code organized and modular.

3 Creating Custom Modules

You can create your own modules by saving Python code in a .py file and importing it into other scripts.

1

Create a file named mymodule.py:

# mymodule.py def say_hello(name): return f"Hello, {name}!"
2

Import and use it in another file:

# main.py import mymodule print(mymodule.say_hello("Alice")) # Output: Hello, Alice!
3

Import specific functions:

from mymodule import say_hello print(say_hello("Bob")) # Output: Hello, Bob!
Pro Tip

Add if __name__ == "__main__": in your module to prevent code from running when imported.

4 The OS Module

The os module provides functions to interact with the operating system, like file management and directory operations.

1

Get current directory:

import os print(os.getcwd()) # Output: Current working directory path
2

List directory contents:

print(os.listdir()) # Output: List of files and folders
3

Create a directory:

os.mkdir("new_folder") print("Directory created!")
Did you know?

Use os.path.join() to create platform-independent file paths.

5 The JSON Module

The json module allows you to work with JSON data, converting Python objects to JSON and vice versa.

1

Convert Python to JSON (dump):

import json data = {"name": "Alice", "age": 25} with open("data.json", "w") as file: json.dump(data, file)
2

Read JSON to Python (load):

with open("data.json", "r") as file: loaded_data = json.load(file) print(loaded_data) # Output: {'name': 'Alice', 'age': 25}
3

String conversion:

json_string = json.dumps(data) print(json_string) # Output: {"name": "Alice", "age": 25}
Pro Tip

Use indent=4 in json.dump() for pretty-printed JSON.
