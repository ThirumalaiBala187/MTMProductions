1 Defining Functions: def and return

Functions in Python allow you to group code into reusable blocks. You define them using the def keyword and can return values with return.

1

Basic function definition:

def greet(): print("Hello, World!")
2

Calling the function:

greet() # Output: Hello, World!
3

Using return:

def add(a, b): return a + b result = add(3, 5) print(result) # Output: 8
Did you know?

If no return is specified, a function returns None by default.

4 Default and Keyword Arguments

Default arguments allow parameters to have preset values, while keyword arguments let you specify arguments by name.

1

Function with default argument:

def greet(name, message="Hello"): print(f"{message}, {name}!") greet("Alice") # Output: Hello, Alice!
2

Overriding default argument:

greet("Bob", "Hi") # Output: Hi, Bob!
3

Using keyword arguments:

greet(message="Hey", name="Charlie") # Output: Hey, Charlie!
Did you know?

Default arguments must come after non-default arguments in the function definition.
